# AWS CloudShell

CloudShell es un servicio de terminal basado en la web que proporciona un entorno de CLI para interactuar con los servicios de AWS. CloudShell viene con la AWS CLI y otras herramientas útiles preinstaladas, lo que facilita a los usuarios ejecutar comandos de AWS sin necesidad de configurar un entorno local.

> La disponibilidad de CloudShell puede variar según la región.

![AWS CloudShell](aws-cloudshell.png)

Desde CloudShell, los usuarios pueden subir y descargar los archivos que se hayan genereado durante la sesión.
