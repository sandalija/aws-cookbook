## IAM Credentials Report

El informe de credenciales de IAM es una herramienta que proporciona un resumen de las credenciales de seguridad asociadas a los usuarios de IAM en una cuenta de AWS. Este informe incluye información sobre contraseñas, claves de acceso, dispositivos MFA y otros detalles relacionados con la seguridad de las cuentas de IAM.

Proprociona una visión general de los accesos de usuario y sus credenciales.

## IAM Access Advisor, IAM Last Access

Desde la vista de un usuario, se puede acceder a la sección de "Access Advisor" para ver los servicios de AWS a los que el usuario ha accedido recientemente. Entre la información se muestra el rol usado por el usuario y la información del último acceso a cada servicio de AWS
