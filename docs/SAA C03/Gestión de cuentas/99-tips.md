## Alias de cuenta

Las cuentas de AWS pueden tener un alias, que es un nombre amigable que se puede usar en lugar del ID de la cuenta para iniciar sesión. El alias de la cuenta se puede configurar en la consola de IAM. El nombre debe ser único en AWS y puede contener letras, números y guiones.

El alias de la cuenta se utiliza para iniciar sesión en la consola de AWS, por ejemplo: `https://my-alias.signin.aws.amazon.com/console/`.
