# Evolución de AWS con el paso de tiempo

Amazon Web Services aparece por primera vez en 2002 de manera interna dentro de Amazon. Durante los siguientes años, su crecimiento es exponencial.

- 2004: Lanzamiento de SQS como el primer servicio de AWS públicamente accesible
- 2006: Relanzamiento con S3, SQS y EC2.
- 2007: Lanzamiento en Europa

## AWS en números

- 31% del mercado en 2023 según Gartner
- 1.000.000 de usuarios activos
