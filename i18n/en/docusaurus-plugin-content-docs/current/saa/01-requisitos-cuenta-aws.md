# Requisitos para la creación de una cuenta de AWS

Los requisitos para crear una cuenta de AWS son
- una direccón de correo
- una tarjeta de pagos

> ⚠️ La dirección de correo no puede estar asociada a una cuenta ya existente. De hecho, aunque se cierre la cuenta, **la dirección de correo no podrá volver a ser asociada**.
